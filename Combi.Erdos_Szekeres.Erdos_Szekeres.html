Combi.Erdos_Szekeres.Erdos_Szekeres : The Erdös-Szekeres theorem

A proof of the Erdös Szekeres theorem about longest increassing and decreassing subsequences. We prove it as a corollary of Greene's theorem on the Robinson-Schensted correspondance.

Require Import mathcomp.ssreflect.ssreflect.

Require Import partition tableau Schensted ordtype Greene Greene_inv.

Set Implicit Arguments.

Open Scope N.

Section OrderedType.

Variable T : inhOrdType.

Lemma Greene_rel_one (s : seq T) (R : rel T) :
  ∃ t : seq T, subseq t s ∧ sorted R t ∧ size t = (Greene_rel R s) 1.

Theorem Erdos_Szekeres (m n : nat) (s : seq T) :
  (size s) > (m × n) →
  (∃ t, subseq t s ∧ sorted leqX t ∧ size t > m) ∨
  (∃ t, subseq t s ∧ sorted gtnX t ∧ size t > n).

End OrderedType.
